Case Study

Budget Tracker App

Project: Budget Tracker App

Challenge: Users could not track expenses clearly and needed a simple app that stores records even after closing the app.

Approach: Built an Android app using Java and SQLite. I implemented create, read, update and delete features so users can manage expenses properly.

Outcome: The app supports full CRUD functionality, SQLite persistence and a clear expense management workflow.

Tools: Java, Android Studio, SQLite, RecyclerView.
