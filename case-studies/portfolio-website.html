Case Study

Developer Portfolio Website

Project: Professional portfolio for recruiters and clients.

Challenge: A normal portfolio only shows projects, but it does not always help visitors take action.

Approach: Added project categories, EmailJS request form, booking links, animated skills, case studies and an assistant widget.

Outcome: Recruiters and clients can view work, understand thinking, request a quote, book a meeting or download a CV.

Tools: HTML, CSS, JavaScript, EmailJS, Calendly, Google Calendar.
